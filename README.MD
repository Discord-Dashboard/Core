# Discord-Dashboard

- Easier creation of a Dashboard for Bot using Node.js
- No need to know how the API works, no need to write it
- Easy configuration
- Community support
- Free for no-profit usage
- Paid for profit usage

# Install

```js
npm i discord-dashboard
//or
yarn add discord-dashboard
```
